A comprehensive, modern classifieds and marketplace application. **FoundIt** is designed to be the ultimate local hub where users can buy, sell, and rent absolutely anything—from real estate and vehicles to electronics, furniture, and local services.

This repository contains the complete frontend and integrated Firebase backend for the main consumer application, featuring a dynamic category system, secure user authentication, and a responsive marketplace feed.

## ✨ Key Features

- **Dynamic Classifieds Categories:** Built to handle diverse listings, including Motors, Property for Rent/Sale, Electronics, Jobs, and Classifieds.
- **Secure Authentication:** Integrated with Firebase for safe and reliable global user accounts.
- **Instant Messaging:** A dedicated chat system allowing buyers and sellers to negotiate securely within the app.
- **Premium Onboarding:** Smooth, animated introductions utilizing Lottie animations and dynamic page indicators.
- **Cross-Platform:** Renders flawlessly on iOS and Android devices.

## 🛠️ Built With

* **Framework:** [Flutter](https://flutter.dev/)
* **Backend:** [Firebase](https://firebase.google.com/) (Auth & Cloud Firestore)
